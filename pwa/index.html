Scoreboard

  • Tap a player's card → +1 point
  • Swipe up on a card → +1 · swipe down → −1
  • Voice: tap 🎙, then every command must include tally: tally blue point, tally another point for red, tally minus red. Wake word required each time to avoid picking up chatter.
  • “to N” selector = when a player reaches N with a 2-point lead, celebrate. Scores keep going.
  • Undo reverses last change · Next starts a new game · Reset clears everything.
  • Voice log: every recognition is stored locally. Tap 📋 to download it as a text file.
  • On iPhone: to install as an app, tap Share in Safari, then Add to Home Screen. Voice needs iOS 14.5+ and Safari (not Chrome).
